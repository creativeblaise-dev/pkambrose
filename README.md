# PKAmbrose Music

Personal website for **Philip K. Ambrose** — bass player & musical director based in London. Features tour dates, bio, photo gallery, equipment rundown, shop, and contact info.

**Live site:** [pkambrose.com](https://pkambrose.com)

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 8** (build tool & dev server)
- **Tailwind CSS 4** (via `@tailwindcss/vite`)
- **Framer Motion** (animations)
- **clsx** + **tailwind-merge** (utility class composition)
- **Netlify** (hosting & deployment)

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command             | Description                  |
| ------------------- | ---------------------------- |
| `npm run dev`       | Start dev server             |
| `npm run build`     | Type-check & build for prod  |
| `npm run preview`   | Preview production build     |
| `npm run lint`      | Run ESLint                   |
